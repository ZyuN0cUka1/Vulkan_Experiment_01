# Requirement
```
Git
CMake:		3.16~
Vulkan SDK:	1.3.250
```
# Installation
## Windows (CMake gui)
- Visual Studio 2022 works better with CMakePreset
## MacOS / Linux
To compile the environment, go into the root directory and enter in a terminal:
```
bash
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ../
make
```
